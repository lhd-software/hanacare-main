# Claude Code Rules

## Quy tắc quan trọng

### KHÔNG TỰ ĐỘNG COMMIT
- **TUYỆT ĐỐI KHÔNG** tự động commit code sau khi hoàn thành task
- Chỉ commit khi người dùng yêu cầu rõ ràng: "commit", "git commit", "vui lòng commit"
- Luôn hỏi lại xác nhận trước khi thực hiện commit

### Quy trình làm việc
1. Hoàn thành task được yêu cầu
2. Hiển thị kết quả/thay đổi
3. **CHỜ** xác nhận từ người dùng trước khi commit
4. Nếu người dùng không yêu cầu commit, không thực hiện

### Mẫu câu hỏi xác nhận trước khi commit:
"Bạn có muốn tôi commit những thay đổi này không?"
"Bạn có muốn tôi lưu code này lên git không?"

**Lưu ý: Quy tắc này có ưu tiên cao nhất và luôn được áp dụng.**
